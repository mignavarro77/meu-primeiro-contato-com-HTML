O HTML e uma linguagem de programação?????

O HTML é uma linguagen de marcação de de texto (Linguagem de Marcação de HiperTexto). E é o que define o significado e a estrutura do conteudo da web. Para nos poder usar o HTML precisamos da sua estrutura basica:

Vamos agora a aprender algumas das tags mais usadas no html.

Tags mais usadas.

Algumas das tags são:

  • < HEAD >
  • < TITLE >
  • < BODY >
  • < H1 > < H2 > < H3 > < H4 > < H5 > e < H6 >
  • < p >
  • < a >
  • < ul >
  • < li >
  • < ima >
Todas essas tags tem uma fumção especifica para fazer a sua marcação no html. Vamos ver para que servem cada uma delas.

Aplicações das tags.

  • < HEAD > - Declara a informação do site na aba do navegador.
  • < TITLE > - Define o titulo.
  • < BODY > - Define o corpo da pagina.
  • < H1 > ate ó < H6 > - Tags para definir titulos e subtitulos de maior a menor.
  • < p > - Para colocar un paragrafo dentro da pagina.
  • < a > - Serve para criar un link para outro site o fonte de informação.
  • < ul > - Criar uma lista não ordenada.
  • < li > - Criara uma lista ordenada.
  • < ima > - Para colocar uma imagen dentro da pagina.
Alem de essas Tags temos outars tambem usadas no desenvolvimento do site com html. Por exemplo
  • < nav >
  • < id >
  • < hr >
  • < u >
  • < i >
